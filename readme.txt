コンパイルコマンド
g++ -o billiards.exe ball2.cpp -lglut -lGLU -lGL -lm

コマンド一覧

スペースキー…自球の発射
wキー…自球がポケットに入ったときのリセット
sキー…カメラの方向移動(左回りに円形)
dキー…カメラの移動停止
rキー…全球の初期位置リセット
qキー…終了コマンド

矢印キー一覧
上キー…ズームアウト
下キー…ズームアップ
左キー…自球の射出方向を左回りに1度ずつ移動(デフォルトは0度でx方向にだけ向かう)
右キー…自球の射出方向を右回りに1度ずつ移動


各関数一覧

オブジェクト描画用
・cylinder…円柱
・tpyramid…角錐体
・cuboid…直方体
・pocket_corner…ポケットまわりのカバー(上手く描画できていない)
・Ground…地面(メッシュ)
・display…ゲームオブジェクトの描画

ゲーム処理用
・Calc_Hit…当たり判定計算
・Hitbox…当たり判定による処理
・Calc_Erase…消去判定計算
・Erasebox…消去判定による処理

その他
idle…待ち時間による判定操作
Init…初期配置,パラメータの決定
command,action…キーボードによる操作
